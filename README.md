## CNplus介绍

<b>CNplus致力于让您轻松地使用中文编码，并提供相关教程。</b>
### 为什么不是用E语言,而是CNplus?
虽然CNplus并没有易语言完善,但**正版CNplus**是免费的,而正版易语言则需要支付200~700CNY后拿到加密狗才能使用正版,相较于易语言,CNplus只需要搭建好python环境,将CNplus.py与您的项目文件置于同一目录即可.
